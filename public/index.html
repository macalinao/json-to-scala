JSON to Scala

Convert JSON to Scala case classes

This tool converts JSON data into a set of Scala case classes. Paste a JSON blob on the left and the equivalent Scala code will be generated on the right, which you can paste into your program. This can easily be used with JSON4S. This tool was written using ScalaJS.

A few caveats:

  • Nulls are handled with the Any type. Fix this with an Option or something equivalent.
  • All numbers use Scala Doubles or Ints. You may want to use some other number type.
  • This does not take the 22 argument limit into account. I haven't personally tried this, but apparently circe can get past this.
